## next14-mongodb

A Next.js v14 app integrated with Mongo database

### Setup

```bash
git clone git@github.com:dhruv-m-patel/next14-mongodb.git
cd next14-mongodb
nvm use
npm ci
cp .env.example .env # Update MONGO_URI
npm run build
npm run dev
```
